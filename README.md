# Allan Pimentel Portfolio

Portfolio moderno e tecnologico em React + Vite, com visual futurista inspirado em infraestrutura, IA e homelab.

## Stack
- React + Vite + TypeScript
- TailwindCSS
- Framer Motion
- React Router

## Estrutura
```
src/
  assets/
    gadgets/
    projects/
  components/
  data/
  pages/
  sections/
  App.tsx
  main.tsx
  index.css
```

## Como rodar
1. Instale dependencias:
```
npm install
```

2. Rode o ambiente de desenvolvimento:
```
npm run dev
```

3. Build de producao:
```
npm run build
```

4. Preview local:
```
npm run preview
```

## Onde editar conteudo
- Dados das skills, projetos e gadgets: `src/data/portfolio.ts`
- Layout principal: `src/pages/Home.tsx`
- Estilos globais e efeitos: `src/index.css`

## Imagens
Os placeholders estao em `src/assets`. Troque pelos seus arquivos reais mantendo os mesmos nomes ou atualize os imports em `src/data/portfolio.ts`.
